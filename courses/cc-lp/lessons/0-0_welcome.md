# 0-0 ワクワクレッスン：LPが勝手にできる世界

このレッスンでは、Claude Codeを使ったLP制作の全体像を体験します。
「話しかけるだけでWebページが作れる」という新しい体験を、まず実感してみましょう。

---

## なぜこれをするのか？

いきなりセットアップや技術的な話から始めると、ゴールが見えなくて不安になりがちです。
まず完成形を見て「こんなことができるんだ」とワクワクすることで、残りの学習がスムーズに進みます。

これができると、今後のレッスンで「何のためにやっているか」が常にイメージできる状態になります。

---

## LP制作のビフォー・アフター

まずは、Claude Codeを使う前と後で何が変わるのかを見てみましょう。

### 従来のLP制作 vs Claude Code LP制作

| Before | After |
|---|---|
| HTML/CSSの知識が必要 | 日本語で「こんなLPを作って」と言うだけ |
| デザイナーやコーダーに依頼（数万円〜） | 自分で数分で生成できる |
| 修正のたびにやり取り発生 | 「ここを変えて」で即反映 |
| テンプレートに縛られる | 自由なデザインを対話で実現 |

---

## よくある失敗パターン

> 「どうせプログラミングの知識がないとダメなんでしょ？」と思って始める前から諦めてしまう
>
> Claude Codeは「プログラミングを代わりにやってくれるツール」です。あなたが必要なのは「何を作りたいか」を言葉にする力だけです。
